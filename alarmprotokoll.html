⚙️ Einmalige Einrichtung: EmailJS-IDs im Code eintragen, damit E-Mails automatisch versendet werden. Anleitung →
EmailJS kostenlos einrichten (5 Minuten):
1. Gehe zu emailjs.com → kostenlosen Account erstellen
2. „Add New Service" → Gmail/GMX verbinden → Service ID kopieren
3. „Email Templates" → neues Template → Body: {{message}} → Template ID kopieren
4. Oben rechts → Account → Public Key kopieren
5. Im Code die drei Werte bei EMAILJS_SERVICE_ID, EMAILJS_TEMPLATE_ID, EMAILJS_PUBLIC_KEY eintragen.
