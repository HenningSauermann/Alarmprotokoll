EmailJS kostenlos einrichten (5 Minuten):
1. Gehe zu emailjs.com → kostenlosen Account erstellen
2. "Add New Service" → Gmail oder GMX → verbinden → Service ID kopieren
3. "Email Templates" → neues Template → Betreff: Neues Alarmprotokoll → Body: {{message}} → Template ID kopieren
4. Oben rechts auf deinen Account → Public Key kopieren
5. Unten im Code die drei Werte eintragen bei EMAILJS_SERVICE_ID, EMAILJS_TEMPLATE_ID, EMAILJS_PUBLIC_KEY
→ Kostenlos bis 200 E-Mails/Monat. Kein Server nötig.
